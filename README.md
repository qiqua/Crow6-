# Crow6 Developer Preview

Crow6 Developer Preview 是一个面向融资演示和早期验证的 AI 编程桌面产品原型。

## 当前版本

第一版实现内容：

- Electron 桌面应用壳
- React + TypeScript + Vite 前端
- TailwindCSS 暗色科技感 UI
- 三栏主工作区
- 项目文件树
- 代码预览
- Diff Viewer 展示
- AI Chat / Agent Console 面板
- Skills / Command Suggestion 演示入口
- 本地目录选择和安全文件树扫描

## 开发运行

```bash
npm install
npm run dev
```

## 构建检查

```bash
npm run build
```
